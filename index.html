Xtal Upgrade Path Viewer — Toram Online

Browsing crystals

  • Use the tabs at the top to switch between crystal slots — Weapon, Armor, Additional, Special, Normal.
  • Toggle Show Enhancers to include or hide enhancer crystals within the same tab.
  • Type in the search box to filter crystals by name.
  • Click a stat chip below the tabs to show only crystals that carry that stat. You can select multiple chips at once.

Reading the graph

  • Arrows show the upgrade direction — from a crystal to what it becomes.
  • Crystals in longer upgrade chains appear first (top of the graph).
Solid node — base crystal
Dashed node — enhancer crystal (same slot, different tier)
Faded dotted — upgrade target from a different slot (shown to keep arrow intact)

Getting details

  • Hover a crystal — tooltip shows all stats and drop sources inline.
  • Click a crystal — side panel shows full details including sell value, process material, and a direct link to Coryn Club.

Navigation

  • Reset View button — bring everything back into view if you get lost.
  • Minimap (bottom-left corner) — overview of the full graph. Click anywhere on it to jump there.
  • Scroll to zoom, drag the background to pan.
